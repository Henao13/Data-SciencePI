Proyecto: EDA pedidos y producción

Este repositorio contiene el notebook `proyect.ipynb`, los datos originales `data/pedidos.csv` y `data/produccion.csv`, y el merged preparado `data/join_pedidos_produccion_unique_left.csv`.

Resumen:
- Notebook: análisis y preparación de datos
- Script de apoyo: `analisis_datasets.py`

Instrucciones rápidas:
1. Crear y activar un entorno virtual (opcional).
2. Instalar dependencias: pandas, numpy, matplotlib, seaborn.
3. Ejecutar el notebook desde JupyterLab o VS Code.
